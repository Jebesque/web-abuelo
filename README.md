# Enrique Delgado Portfolio (ENDELCON)

Modern bilingual (ES/EN) artist portfolio built with **React + Vite + Tailwind CSS**.

## Features
- Premium contemporary gallery style
- Bilingual UI with ES/EN language toggle
- Dedicated sections: Home, About, Biography, Works, Recent Works, Watercolours, Previous Works, Exhibitions, Press Reviews, Videos, Contact
- Reusable component architecture
- Data-driven artworks and exhibitions
- Placeholder artwork images ready to swap with final paintings
- Contact form layout (frontend only)

## Stack
- React 18
- Vite 5
- Tailwind CSS 3

## Development
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
npm run preview
```

## Deploy
This project is ready for deployment on Vercel or GitHub Pages with static hosting.
